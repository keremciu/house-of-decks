# node against humanity

Made for fun! Online version of [Cards Against Humanity](https://cardsagainsthumanity.com/).

### Commands

I used `yarn` as package manager for this project.

1. `yarn & (cd client && yarn)`
2. `yarn start`

#### For development

`yarn dev`
